## What I Learned

- In baah scripting, there are three streams
- STDIN (standard input) A stream of data into the program
- STDOUT (standard output) A stream of data out of the program
- STDERR (standard error) Errors and exceptions are  written to.
- cat animals.txt | cut -d "," -f 2 | sort
- This means the standard output STDOUT of cat animals.txt  becomes the standard input STDIN for the cut command and the standard output of the cut command becomes the standard input to the sort command
- Another example cat sports.txt 1> new_sports.txt. 1 means standard output
