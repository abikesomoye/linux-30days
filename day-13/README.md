## What I Learned

- ? matches a single character eg 201?.txt will match 2016.txt, 2017.txt 2018.txt but not 2020.txt or even 2017-01.txt
- [...] will match any one of the characters inside the square brackets eg 201[78].txt matches 2017.txt and 2018.txt
- {...} matches any comma-separated pattern eg {*.txt,*.csv}
